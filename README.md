# Sadat Hossain — Personal Website

Static GitHub Pages website for Sadat Hossain.

## How to use

1. Create a GitHub repository named either:
   - `Sadat75.github.io` for a user site, or
   - any repository name, then enable GitHub Pages from repository settings.
2. Upload all files from this folder to the repository.
3. Replace the placeholder profile image:
   - Put your photo at `assets/img/profile.jpg`
   - Recommended: square image, at least 600×600 px.
4. Replace the CV file if needed:
   - Put your updated CV at `assets/pdf/Sadat_Hossain_CV.pdf`
5. Enable GitHub Pages:
   - Settings → Pages → Deploy from branch → `main` → `/root`.

## Editing

Most website content is inside `index.html`. Styling is inside `assets/css/style.css`.
